01 — Foundations

Principles

Four rules extracted from the live site that keep the system coherent. Dark canvas, bright point of focus, quiet chrome.

01

Dark is the default

The site opens dark and stays dark. Light is a courtesy, not a baseline.

02

One glowing accent

Cyan #00F1F1 is the only hue with saturation. Use it to point, never to decorate.

03

Foreground by opacity

Twelve opacity steps of the foreground replace a grey scale. Hierarchy by alpha, not hue.

04

Motion scales with scroll

Transitions use --motion-scale so fast readers get snappy UI, slow readers get soft.

02 — Typography

Type system

Montserrat carries everything, using the full 100–900 weight range for hierarchy. JetBrains Mono is added for tokens, metadata, and code — a supporting voice, never decorative.

05 — Components

Core library

The pieces used across the home page, case studies, and template sub-sites. All rebuilt in Figma with the tokens above.

Buttons

Primary

Filled cyan. Hover: translateY(-2px) scale(1.03) + halo brightens to rgba(accent, 0.60). Transition: 150ms ease.

Secondary

Border + transparent. Hover: accent border, accent text, soft glow + lift.

Ghost

Text only. Hover: accent colour + 1px lift. Used for low-emphasis actions in forms and modals.

Accent Outline

Accent border + 10% accent fill. Hover: 20% fill, border rgba(accent, 0.65), strong glow + lift. Used for primary CTAs on dark sections (hero, mentoring).

Danger

Red outline. Hover: 12% red fill + red glow + lift. Reserved for destructive, irreversible actions only.

Icon

44×44 circle · icon only. Hover: scale(1.12) + accent ring + glow. Used for carousel nav, social links, toolbars.
